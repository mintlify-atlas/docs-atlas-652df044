The ElevenLabs integration generates professional voice narration for the Dream Awakening phase, creating audio announcements of the winning agent.

## Overview

The module handles:
- Text-to-speech conversion using ElevenLabs API
- Cloned voice support for personalized narration
- Winner announcement script generation
- Audio file management (MP3 format)
- Voice configuration and quality settings

<Note>
  Requires an ElevenLabs API key. Sign up at [elevenlabs.io](https://elevenlabs.io) to get started.
</Note>

## Core Functions

### generate_audio

Convert text to speech using ElevenLabs API.

```python
def generate_audio(
    text: str,
    output_path: Optional[str] = None,
    voice_id: Optional[str] = None,
) -> Optional[bytes]:
```

#### Parameters

<ParamField path="text" type="string" required>
  The text to convert to speech. Can be multi-line with natural pauses.
</ParamField>

<ParamField path="output_path" type="string">
  Optional path to save the generated MP3 file. Parent directories are created automatically.
</ParamField>

<ParamField path="voice_id" type="string">
  Optional voice ID to use. Overrides `ELEVENLABS_CLONED_VOICE_ID` env var.
  
  Defaults to Rachel (ID: `EXAVITQu4vr4xnSDxMaL`) if not provided.
</ParamField>

#### Returns

<ResponseField name="audio_bytes" type="bytes | None">
  MP3 audio data as bytes if successful, `None` if API key not configured or request failed
</ResponseField>

#### Example

<CodeGroup>
```python Basic Usage
from src.elevenlabs import generate_audio

audio = generate_audio(
    text="The winner is Agent Gamma with 92.5 points.",
    output_path="outputs/announcement.mp3",
)

if audio:
    print(f"Generated {len(audio)} bytes of audio")
else:
    print("Failed to generate audio")
```

```python Custom Voice
audio = generate_audio(
    text="Welcome to the Dream Foundry!",
    voice_id="21m00Tcm4TlvDq8ikWAM",  # Rachel's voice ID
    output_path="outputs/intro.mp3",
)
```

```python In-Memory Usage
# Generate without saving to disk
audio_bytes = generate_audio(
    text="Processing complete.",
)

if audio_bytes:
    # Stream or process in memory
    import io
    audio_file = io.BytesIO(audio_bytes)
```
</CodeGroup>

## Voice Configuration

The module uses the following voice settings for optimal narration quality:

<CodeGroup>
```json Voice Settings
{
  "model_id": "eleven_monolingual_v1",
  "voice_settings": {
    "stability": 0.5,
    "similarity_boost": 0.75
  }
}
```
</CodeGroup>

### Voice Parameters

<ParamField path="model_id" type="string" default="eleven_monolingual_v1">
  The ElevenLabs model to use. `eleven_monolingual_v1` provides high-quality English narration.
</ParamField>

<ParamField path="stability" type="float" default="0.5">
  Controls voice consistency (0.0-1.0). Higher = more consistent, lower = more expressive.
</ParamField>

<ParamField path="similarity_boost" type="float" default="0.75">
  How closely to match the original voice (0.0-1.0). Higher = closer match to cloned voice.
</ParamField>

### Custom Voices

To use a cloned voice:

1. Visit [ElevenLabs Voice Lab](https://elevenlabs.io/voice-lab)
2. Create or clone a voice
3. Copy the Voice ID
4. Add to `.env`:
   ```bash
   ELEVENLABS_CLONED_VOICE_ID=your_voice_id_here
   ```

## Error Handling

<CodeGroup>
```python Configuration Errors
if not api_key:
    return None  # Silently return None if not configured
```

```python API Errors
if response.status_code != 200:
    print(f"[ElevenLabs] Error: {response.status_code} - {response.text[:200]}")
    return None
```

```python Network Errors
try:
    response = requests.post(url, json=payload, timeout=30)
except Exception as e:
    print(f"[ElevenLabs] Exception: {e}")
    return None
```
</CodeGroup>

<Note>
  The module uses graceful degradation - if ElevenLabs is not configured or fails, it returns `None` rather than raising exceptions.
</Note>

## Usage Patterns

<CodeGroup>
```python Pattern: Generate and Save
# Most common: generate and save to disk
audio = generate_audio(
    text=script,
    output_path="announcement.mp3"
)

if audio:
    # File is saved, audio bytes also available
    pass
```

```python Pattern: Generate Only
# Generate without saving (for streaming/processing)
audio_bytes = generate_audio(text=script)

if audio_bytes:
    # Process in memory
    upload_to_s3(audio_bytes)
```

```python Pattern: Script + Audio
# Common workflow: generate script then audio
script = generate_winner_script(
    winner_name="Agent Alpha",
    winner_score=88.0,
    objective="Build feature X",
)

audio = generate_audio(
    text=script,
    output_path="winner.mp3",
)
```

```python Pattern: One-Shot Winner
# Simplest: use convenience function
audio = generate_winner_audio(
    winner_name="Agent Beta",
    winner_score=95.0,
    objective="Build feature Y",
    output_path="beta_wins.mp3",
)
```
</CodeGroup>
